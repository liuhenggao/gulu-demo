## 开始使用
1. 添加 CSS 样式
使用本框架前，前请先开启border-box

```css
*,*::before,*::after{
    box-sizing: border-box;
}
```
IE 8 及以上浏览器都支持此样式

你还需要设置默认颜色等变量 （后续会改为SCSS变量）
```css
:root {
    --font-size: 14px;
    --button-height: 32px;
    --button-bg: white;
    --button-active-bg: #eee;
    --border-radius: 4px;
    --color: #333;
    --border-color: #999;
    --border-color-hover: #666;
}
```
IE 15 及以上浏览器支持此样式

2. 安装 gulu-demo
   ```
    npm i --save gulu-demo-1995
   ```
